## Description
Bacchusはお酒限定の投稿サービスである。</br>
プログラミング学習のアウトプットとして今回フレームワークを使用せずフルスクラッチでのWebサービス開発を行った。</br>

全ての機能を一から開発することによってWebサービスに基本的に備わっている機能を実装する力などが身に付いた。</br>
また、デザインカンプもXDを使って一からデザインしました。</br>
学習開始1ヶ月半から初めて、制作期間は約1ヶ月です。
